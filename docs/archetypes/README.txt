Farinhas Cósmicas V3.4 — NO-CDN (ASCII-only)

• Removidos shaders e CDNs (sem Three.js, sem overlay). 100% DOM+CSS+JS leve.
• Círculo tracejado girando + órbita primária ASCII (glifo único por arquétipo).
• Tri-Ring ASCII puro com densidade/velocidade únicas por arquétipo.
• Faces centrais com frames ASCII (3–5), sem label e sem "blink".
• Fundo transparente; `?bg=dark` aplica um fundo escuro (#0b1e3b).
• Safe mode: `?safe=1` desacelera tudo.

Customização rápida: frames no JS (`frames`), velocidade `base` (ms), vars CSS (`--r*`, `--speed*`, `--n*`, `--color`, `--faceSize`).
